=== Getting Started with a Use Case

Clone the repository and pick any use case:

[source,bash]
----
git clone https://github.com/ArcadeData/arcadedb-usecases.git
cd arcadedb-usecases/<use-case-name>
docker compose up -d
./setup.sh
./queries/queries.sh
----

=== Planned Use Cases

The following use cases are in development:

* *Semantic Search* — Standalone vector search for e-commerce and document retrieval (Python)
* *Geospatial Analytics* — Fleet tracking and location-based services (Python, JavaScript)
* *Content Management* — CMS with documents, full-text search, and graph taxonomies (JavaScript/TypeScript)
* *Network Monitoring* — Network topology with time-series metrics and Grafana (Python)
* *Data Lineage* — Data pipeline lineage tracking with graph traversal (Python)
