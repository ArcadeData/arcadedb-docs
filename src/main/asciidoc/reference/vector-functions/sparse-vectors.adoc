[[vector-sparse]]
==== Sparse Vectors

Functions for working with sparse vector representations. Sparse vectors store only non-zero elements, making them efficient for high-dimensional data with many zero values (e.g., TF-IDF, bag-of-words).

[cols="2,3,1"]
|===
| Function | Description | Example

| `vectorSparseCreate(indices, values, size)` | Create a sparse vector from index and value arrays. | `vectorSparseCreate([0,2], [0.5,0.3], 5)`
| `vectorDenseToSparse(v, threshold)` | Convert a dense vector to sparse, dropping values below threshold. | `vectorDenseToSparse([0.5,0.0,0.1], 0.05)`
| `vectorSparseToDense(sv)` | Convert a sparse vector back to dense. | `vectorSparseToDense(sparseVec)`
| `vectorSparseDot(sv1, sv2)` | Dot product of two sparse vectors. | `vectorSparseDot(s1, s2)`
| `vectorSparsity(v, threshold)` | Fraction of values below threshold. | `vectorSparsity(emb, 0.01)`
|===

See <<sql-functions,SQL Functions Reference>> for full syntax.
