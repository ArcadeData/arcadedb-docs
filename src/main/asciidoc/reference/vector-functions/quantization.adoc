[[vector-quantization-functions]]
==== Quantization

Functions for compressing and decompressing vectors. Use these for manual quantization workflows; for automatic quantization, set the `quantization` parameter on the index (see <<vector-embeddings-howto,Vector Embeddings How-To>>).

[cols="2,3,1"]
|===
| Function | Description | Example

| `vectorQuantizeInt8(v)` | Quantize float vector to INT8 (4x compression). | `vectorQuantizeInt8(embedding)`
| `vectorQuantizeBinary(v)` | Quantize float vector to binary (32x compression). | `vectorQuantizeBinary(embedding)`
| `vectorDequantizeInt8(v, min, max)` | Restore INT8 vector to float approximation. | `vectorDequantizeInt8(qv, 0.0, 1.0)`
| `vectorApproxDistance(qv1, qv2, type)` | Approximate distance between quantized vectors. | `vectorApproxDistance(q1, q2, 'INT8')`
|===

See <<sql-functions,SQL Functions Reference>> for full syntax.
