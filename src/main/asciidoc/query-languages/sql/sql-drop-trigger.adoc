==== SQL - `DROP TRIGGER`
image:../images/edit.png[link="https://github.com/ArcadeData/arcadedb-docs/blob/main/src/main/asciidoc/query-languages/sql/sql-drop-trigger.adoc" float=right]

NOTE: Available from version 26.2.1

Removes a trigger from the database.

*Syntax*

[source,sql]
----
DROP TRIGGER [IF EXISTS] <trigger-name>
----

* *`IF EXISTS`* Optional: do not fail if the trigger does not exist
* *`<trigger-name>`* The name of the trigger to remove

*Examples*

* Drop a trigger:

[source,sql]
----
DROP TRIGGER user_audit;
----

* Drop a trigger only if it exists:

[source,sql]
----
DROP TRIGGER IF EXISTS old_trigger;
----

For more information about triggers, see <<sql-triggers,Triggers>>.
