[[vector-functions]]
=== Vector Functions

ArcadeDB provides 40+ SQL functions for vector operations. Functions are available in two naming forms that work identically:

* Legacy: `vectorNeighbors()`, `vectorCosineSimilarity()`, etc.
* Namespaced: `vector.neighbors()`, `vector.cosinesimilarity()`, etc.

The namespaced form is recommended for new code. Full syntax and examples for each function are in the <<sql-functions,SQL Functions Reference>>.
