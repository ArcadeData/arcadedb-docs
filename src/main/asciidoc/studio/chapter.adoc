[[Studio]]
== Studio

Studio is the web tool that comes in bundle with ArcadeDB Server. It starts automatically on port 2480. If you have installed ArcadeDB, and it's running on your local computer, then you can access Studio at http://localhost:2480. Replace "localhost" with the host name or IP address of the server where ArcadeDB server is running.
