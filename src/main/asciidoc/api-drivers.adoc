[[api-drivers]]
= API & Drivers
:description: ArcadeDB connectivity reference -- supported languages and wire protocols.

++++
<section class="docs-hero docs-hero-compact docs-hero-with-summary">
  <div class="docs-hero-summary">
    <p><strong>Connect from any language, over the protocol of your choice.</strong> ArcadeDB exposes Neo4j BOLT, the PostgreSQL wire protocol, MongoDB protocol, Redis protocol, the HTTP/JSON API, gRPC, an MCP server, and a native Java embedded API &mdash; so most stacks already have a driver that just works.</p>
  </div>
</section>
