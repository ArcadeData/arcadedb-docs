= Use Cases

Explore real-world applications built with ArcadeDB. Each use case includes a complete, runnable project with Docker Compose, sample data, and queries in multiple languages.

Full source code: https://github.com/ArcadeData/arcadedb-usecases[ArcadeDB Use Cases Repository]

[%header,cols="30%,40%,30%"]
|===
| Use Case | Description | Features
| https://github.com/ArcadeData/arcadedb-usecases/tree/main/recommendation-engine[Recommendation Engine] | Product recommendations via collaborative filtering and vector similarity | Graph, Vectors, Time-series
| https://github.com/ArcadeData/arcadedb-usecases/tree/main/knowledge-graphs[Knowledge Graphs] | Academic research graph with co-authorship and citation networks | Graph, Vectors, Full-text, Time-series
| https://github.com/ArcadeData/arcadedb-usecases/tree/main/graph-rag[Graph RAG] | Retrieval-augmented generation with knowledge graphs and vector search | Graph, Vectors, Full-text, Bolt, LangChain4j
| https://github.com/ArcadeData/arcadedb-usecases/tree/main/fraud-detection[Fraud Detection] | Multi-signal fraud detection unifying graph, vector, and time-series | Graph, Vectors, Time-series, Cypher
| https://github.com/ArcadeData/arcadedb-usecases/tree/main/realtime-analytics[Realtime Analytics] | IoT and service monitoring with Grafana dashboards | Time-series, Graph, Cypher, PromQL
| https://github.com/ArcadeData/arcadedb-usecases/tree/main/social-network-analytics[Social Network Analytics] | Social analytics with materialized view dashboards | Materialized views, Graph, Time-series
| https://github.com/ArcadeData/arcadedb-usecases/tree/main/supply-chain[Supply Chain] | Multi-tier supply chain visibility and traceability | Graph, Vectors, Time-series, PostgreSQL, JavaScript
| https://github.com/ArcadeData/arcadedb-usecases/tree/main/iam[Identity & Access Management] | Permission resolution through group/role hierarchies | Graph, Time-series, Vectors, PostgreSQL, Python

== Getting Started with a Use Case

Clone the repository and pick any use case:

[source,bash]
----
git clone https://github.com/ArcadeData/arcadedb-usecases.git
cd arcadedb-usecases/<use-case-name>
docker compose up -d
./setup.sh
./queries/queries.sh
----
