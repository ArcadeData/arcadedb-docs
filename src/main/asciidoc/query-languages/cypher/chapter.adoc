[[cypher]]
== Cypher
image:../images/edit.png[link="https://github.com/ArcadeData/arcadedb-docs/blob/main/src/main/asciidoc/query-languages/cypher/chapter.adoc" float=right]

https://opencypher.org/[Cypher] is a declarative graph query language originally developed by Neo4j and now an open standard.
ArcadeDB provides native support for Cypher through its new OpenCypher implementation, allowing you to query and manipulate graph data using familiar Cypher syntax.

*Overview Clauses*

[%header,cols=3]
|===
| Read | Write | Query Composition
| <<cypher-match,MATCH>> | <<cypher-create,CREATE>> | <<cypher-with,WITH>>
| <<cypher-optional-match,OPTIONAL MATCH>> | <<cypher-merge,MERGE>> | <<cypher-unwind,UNWIND>>
| <<cypher-where,WHERE>> | <<cypher-set,SET>> | <<cypher-order-by,ORDER BY>>
| <<cypher-return,RETURN>> | <<cypher-delete,DELETE>> | <<cypher-skip-limit,SKIP/LIMIT>>
|===
