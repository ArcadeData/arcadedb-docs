[[vector-distance-similarity]]
==== Distance & Similarity

Functions for measuring how close two vectors are in the embedding space.

[cols="2,3,1"]
|===
| Function | Description | Example

| `vectorNeighbors(index, vector, k)` | Find k approximate nearest neighbors using the vector index. Core search function. | `vectorNeighbors('Doc[emb]', [0.1,0.2], 5)`
| `vectorCosineSimilarity(v1, v2)` | Cosine similarity between two vectors (-1 to 1). | `vectorCosineSimilarity(a.emb, b.emb)`
| `vectorDotProduct(v1, v2)` | Dot product of two vectors. | `vectorDotProduct(a.emb, b.emb)`
| `vectorL2Distance(v1, v2)` | Euclidean (L2) distance between two vectors. | `vectorL2Distance(a.emb, b.emb)`
| `vectorL1Norm(v)` | Manhattan (L1) norm of a vector. | `vectorL1Norm(embedding)`
| `vectorLInfNorm(v)` | Maximum (L-infinity) norm of a vector. | `vectorLInfNorm(embedding)`
| `vectorMagnitude(v)` | Euclidean (L2) norm of a vector. | `vectorMagnitude(embedding)`
|===

See <<sql-functions,SQL Functions Reference>> for full syntax.
