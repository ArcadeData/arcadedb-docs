=== Graph Algorithms

ArcadeDB includes 68 graph algorithms available as Cypher procedures and SQL functions. Algorithms are organized by category:

* <<algo-dijkstra,Path Finding>> -- Dijkstra, A*, Bellman-Ford, BFS, DFS, K-Shortest Paths, and more
* <<algo-page-rank,Centrality>> -- PageRank, Betweenness, Closeness, Eigenvector, HITS, Katz, and more
* <<algo-louvain,Community Detection>> -- Louvain, Leiden, Label Propagation, SCC, WCC, and more
* <<algo-scc,Structural Analysis>> -- Articulation Points, Bridges, Cliques, K-Core, K-Truss, and more
* <<algo-jaccard,Similarity & Link Prediction>> -- Jaccard, Adamic-Adar, SimRank, Preferential Attachment
* <<algo-node2vec,Node Embedding>> -- Node2Vec, FastRP, GraphSAGE, HashGNN
* <<algo-max-flow,Network Flow>> -- Maximum Flow
* <<algo-random-walk,Traversal & Sampling>> -- Random Walk, Topological Sort
* <<algo-graph-summary,Graph Statistics>> -- Graph Summary
