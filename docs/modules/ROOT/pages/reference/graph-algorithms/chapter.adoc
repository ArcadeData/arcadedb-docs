= Graph Algorithms

ArcadeDB includes 68 graph algorithms available as Cypher procedures and SQL functions. Algorithms are organized by category:

* xref:reference/graph-algorithms/algorithms.adoc#algo-page-rank[Centrality] -- PageRank, Betweenness, Closeness, Eigenvector, HITS, Katz, and more
* xref:reference/graph-algorithms/algorithms.adoc#algo-louvain[Community Detection] -- Louvain, Leiden, Label Propagation, SCC, WCC, and more
* xref:reference/graph-algorithms/algorithms.adoc#algo-scc[Structural Analysis] -- Articulation Points, Bridges, Cliques, K-Core, K-Truss, and more
* xref:reference/graph-algorithms/algorithms.adoc#algo-jaccard[Similarity & Link Prediction] -- Jaccard, Adamic-Adar, SimRank, Preferential Attachment
* xref:reference/graph-algorithms/algorithms.adoc#algo-node2vec[Node Embedding] -- Node2Vec, FastRP, GraphSAGE, HashGNN
* xref:reference/graph-algorithms/algorithms.adoc#algo-max-flow[Network Flow] -- Maximum Flow
* xref:reference/graph-algorithms/algorithms.adoc#algo-random-walk[Traversal & Sampling] -- Random Walk, Topological Sort
