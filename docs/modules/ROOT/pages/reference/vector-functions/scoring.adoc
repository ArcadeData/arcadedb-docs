[[vector-scoring]]
= Scoring & Ranking

Functions for combining multiple search signals into unified scores. Essential for hybrid search (vector + full-text) and multi-vector ranking.

[cols="2,3,1"]
|===
| Function | Description | Example

| `vectorHybridScore(score1, score2, alpha)` | Weighted combination of two scores. `alpha` controls the blend (0.0 = all score2, 1.0 = all score1). | `vectorHybridScore(vecScore, ftScore, 0.7)`
| `vectorRRFScore(rank1, rank2, ..., k)` | Reciprocal Rank Fusion across multiple ranked lists. Last argument is the k parameter (default 60). | `vectorRRFScore(r1, r2, 60)`
| `vectorMultiScore(scores, method, weights)` | Combine multiple scores using MAX, MIN, AVG, or WEIGHTED_AVG. | `vectorMultiScore([0.9,0.7], 'MAX')`
| `vectorNormalizeScores(scores)` | Normalize a score vector to [0,1] range. | `vectorNormalizeScores([1.0, 2.0, 3.0])`
| `vectorScoreTransform(score, fn)` | Apply a transformation (SIGMOID, LOG, SQRT, LINEAR). | `vectorScoreTransform(0.5, 'SIGMOID')`
|===

See xref:reference/sql/sql-functions.adoc#sql-functions[SQL Functions Reference] for full syntax.
