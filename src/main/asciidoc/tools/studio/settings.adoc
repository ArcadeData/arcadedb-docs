[[studio-settings]]
==== Settings Panel

The *Settings* panel controls Studio's own appearance.
It is pinned at the bottom of the icon sidebar.

[[studio-settings-appearance]]
===== Appearance

The *Theme* control offers three options:

* *Light* — light theme.
* *Dark* — dark theme.
* *System* — follow the operating-system theme; Studio listens for OS changes and switches on the fly.

The choice is stored in browser local storage (`studio-theme`) and persists across sessions.

Server- and database-scoped configuration lives elsewhere: see <<studio-server-settings,Server › Server Settings>> and <<studio-database-settings,Database › Database Settings>>.
