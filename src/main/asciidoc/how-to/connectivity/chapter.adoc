* <<python,Python>> - HTTP, PostgreSQL, and embedded bindings
* <<postgres-driver,Postgres Protocol Plugin>> - PostgreSQL wire protocol support

* <<http-api-csharp,C#/.NET (HTTP/JSON)>> - Connect from .NET applications
* <<http-api-elixir,Elixir (HTTP/JSON)>> - Connect from Elixir applications
* <<http-api-nodejs,Node.js / JavaScript>> - Connect from Node.js applications
