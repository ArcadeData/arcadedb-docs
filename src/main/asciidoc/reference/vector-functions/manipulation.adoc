[[vector-manipulation]]
==== Manipulation & Properties

Functions for transforming vectors and inspecting their properties.

[cols="2,3,1"]
|===
| Function | Description | Example

| `vectorNormalize(v)` | Normalize to unit length (L2 norm = 1). | `vectorNormalize(embedding)`
| `vectorAdd(v1, v2)` | Element-wise addition. | `vectorAdd(a, b)`
| `vectorSubtract(v1, v2)` | Element-wise subtraction. | `vectorSubtract(a, b)`
| `vectorScale(v, scalar)` | Multiply each element by a scalar. | `vectorScale(emb, 2.0)`
| `vectorClip(v, min, max)` | Clamp values to a range. | `vectorClip(emb, 0.0, 1.0)`
| `vectorDimension(v)` | Number of dimensions in the vector. | `vectorDimension(embedding)`
| `vectorIsNormalized(v, tol)` | Check if vector is unit length. | `vectorIsNormalized(emb, 0.01)`
| `vectorHasNaN(v)` | Check for NaN values. | `vectorHasNaN(embedding)`
| `vectorHasInf(v)` | Check for infinity values. | `vectorHasInf(embedding)`
| `vectorToString(v)` | Convert vector to string representation. | `vectorToString(embedding)`
|===

**Aggregation functions** (operate across rows):

[cols="2,3,1"]
|===
| Function | Description | Example

| `vectorAvg(field)` | Element-wise average across rows. | `vectorAvg(embedding)`
| `vectorMax(field)` | Element-wise maximum across rows. | `vectorMax(embedding)`
| `vectorMin(field)` | Element-wise minimum across rows. | `vectorMin(embedding)`
| `vectorSum(field)` | Element-wise sum across rows. | `vectorSum(embedding)`
|===

**Statistics:**

[cols="2,3,1"]
|===
| Function | Description | Example

| `vectorVariance(v)` | Variance of vector components. | `vectorVariance(embedding)`
| `vectorStdDev(v)` | Standard deviation of vector components. | `vectorStdDev(embedding)`
|===

See <<sql-functions,SQL Functions Reference>> for full syntax.
