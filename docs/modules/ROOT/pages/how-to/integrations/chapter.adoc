= BI and Analytics Integration

ArcadeDB integrates with popular BI and analytics tools through its xref:how-to/connectivity/postgres.adoc#postgres-driver[PostgreSQL wire protocol] and dedicated connectors.
Since ArcadeDB speaks the PostgreSQL protocol on port 5432, virtually any tool that supports PostgreSQL can connect without a custom driver.

All tools listed below connect via ArcadeDB's PostgreSQL wire protocol unless stated otherwise.
See xref:how-to/connectivity/postgres.adoc#postgres-driver[Postgres Protocol Plugin] for setup instructions.
