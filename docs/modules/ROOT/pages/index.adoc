= ArcadeDB Documentation
:description: Antora skeleton — Phase 1 of the documentation redesign.

This is the Antora preview build for the ArcadeDB documentation redesign.

The current production documentation continues to live at https://docs.arcadedb.com[docs.arcadedb.com] and is built from the `main` branch via the existing Maven + Asciidoctor pipeline.

== Status

[cols="1,3"]
|===
| Phase | Description

| 1 | Antora skeleton (this build) — playbook, component, preview deploy
| 2 | Migrate content from `src/main/asciidoc/` into `docs/modules/ROOT/pages/`, rewrite cross-references
| 3 | Custom UI bundle (3-column layout, search, brand)
| 4 | Cut over `docs.arcadedb.com` to Antora and retire the old pipeline
|===

== Local build

[source,shell]
----
npm ci
npm run build
npm run serve
----

Then open http://localhost:8080.
